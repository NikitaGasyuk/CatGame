# Игра про путишествие кота в мире

Простая игра на с++

У неё нет смысла, просто указывается размер мира в котором будет гулять кот. Некоторые обозначения. 0 -> Кот просто прошел, ничего не нашел 

1 -> Кот нашел\встретил то, что сделало его счастливым, ну например колбаску. За счёт этого повысилось настроение кота 

-1 -> Кот нашел\встретил то, что принесет беду, кот теряет жизнь. Ну например плохо закрепленный шкаф, который кот сам на себя обранил.

В конце выведится полный путь пройденный котом.
